# Blog Site

A mock blog site to put to use the tech stack that I've learnt in my internship. Displays blog posts on the main page.
Allows for user to add posts and edit existing ones.
There is also a Jobs Page using HackerNews API

## Getting Started

### Prerequisites

You'll need Node installed.
You'll also need either MongoDB Atlas (cloud) or MongoDB Compass (local)

### Environment variables

- PORT
  - The port the server will run on
- mongoURI
  - The MongoDB token.

### Installing

Use nodemon to run both server and client together from the top of the folder:

    npm run start

## Deployment

Currently Deployed on Heroku.
The entry-point for the application is `client/dist/...`.
